# CS517_DecToAny
This program converts decimals in the range [0, 1) to base any.

- Run `python3 convert_dec_to_any.py 60 0.5 0.25 0.75 0.8 0.16666`

The end result will be in the form:
| Base 10 | Base 60 |
| :-------|:--------|
| 0.5     | 0.30    |
| 0.25    | 0.15    |
| 0.75    | 0.45    |
| 0.8     | 0.48    |
| 0.16666 | 0.10    |
